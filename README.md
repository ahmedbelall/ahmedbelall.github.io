A professional website showcasing comprehensive web development services by Ahmed Belal, with full support for Arabic and English languages.

## 🚀 الاستخدام | Usage

### تبديل اللغة | Language Toggle
اضغط على الزر "English" أو "العربية" في الأعلى لتبديل اللغة بسهولة.

Click the "English" or "العربية" button at the top to easily switch languages.

### التخصيص | Customization
يمكنك تعديل:
- الألوان من خلال CSS variables في `:root`
- النصوص والمحتوى مباشرة في HTML
- الروابط والأسعار بحسب احتياجاتك
